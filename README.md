2. 유튜브 플레이어 재생하기 
- 유튜브 플레이를 앱 내에서 보여주는 뷰 입니다.
- [YoutubePlayer-in-WKWebView](https://github.com/hmhv/YoutubePlayer-in-WKWebView)와 스냅킷을 사용하여 구현했습니다.
- Youtube동영상 키를 얻는 방법 : 원하는 유튜브 url에서 ```watch?v=```뒤에 있는 요소가 ID가 된다.
